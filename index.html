🎯 위치 × 매출 4사분면 분석

위치 등급(가로) × 매출(세로)로 4분면 분류 — B사분면(상품 교체)과 C사분면(위치 이동)이 핵심 액션 대상

사분면별 액션 가이드

🟥 A: 황금자리 (좋은 위치 + 높은 매출)
→ 현재 베스트 — 유지 + 확대

🟧 B: 활용 부족 (좋은 위치 + 낮은 매출) ⭐
→ 상품 교체! 좋은 자리에 안 팔리는 상품 있음

🟩 C: 숨은 보석 (나쁜 위치 + 높은 매출) ⭐
→ 위치 이동! 잘 팔리는데 자리가 묻혀있음

⬛ D: 데드존 (나쁜 위치 + 낮은 매출)
→ 컷오프 후보 또는 다른 카테고리로 변경

🎯 카테고리별 적정 스타일 제안 다른 매장에 적용 가능

비스터 매장의 회전율을 벤치마크로 — 잘 팔린 카테고리는 스타일 확장, 안 팔린 건 축소
